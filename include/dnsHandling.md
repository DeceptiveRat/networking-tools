## 2. returnUDPListeningSocket
### Synopsis
	int returnUDPListeningSocket(int* IPv4\_listening\_socket, int* IPv6\_listening\_socket)

### Description
	Returns a listening socket for each IP version.

	Both sockets are:
	(1) set to non-block
	(2) bound to localhost

### Return Value
	-1 on error, 0 on success.
