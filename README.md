# Simple Packet Sniffer
## 1 Description
Toy project to help myself learn/practice:
- socket programming
- internet protocols and their fields

## 2 Usage
Navigate to the Packet Sniffer directory

```bash 
./compile.sh
sudo ./packetSniffer
```
to run the sniffer
